# Governance

## Purpose

MOSAIC exists to improve the visibility, discoverability, and reuse of Moroccan open-source AI.

## Roles

### Maintainers
Maintainers can:
- review submissions
- update taxonomy files
- normalize metadata
- manage releases and workflows

### Contributors
Contributors can:
- submit projects
- report errors
- suggest taxonomy improvements
- improve documentation

## Decision making

Routine decisions are handled by maintainers.

Larger changes, such as schema changes or major taxonomy changes, should be discussed through issues or pull requests.

## Scope

MOSAIC is ecosystem-wide. It may include projects from:
- universities
- startups
- companies
- communities
- nonprofits
- research labs
- independent contributors

## Neutrality

Inclusion in the catalog does not imply endorsement.
